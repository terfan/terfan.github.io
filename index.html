    PROJECTS

  • Charity Juke

    Charity Juke is a digital jukebox, where event coordinators can set up a playlist for an event and guests can pay a specified amount to add a song, with the intention of the proceeds going to charity. It uses Facebook for login, the Soundcloud API for searching tracks, and Venmo for payments. We used Bootstrap in the front end and Firebase in the back end. I worked on the song search, music player, and payments. This was created for PennApps Fall 2015 in a team of 4.

  • UpSpire

    UpSpire is a place where people can go to see content curated for their current mood, added by other members of the community. This was created for Dropbox Summer Hack Day 2015 in a team of 5. I created the front page and designed the overall look and feel of the site. We also won the award for best teamwork!

  • Google Translate Chrome Extension 2.0

    My project (shared with my EP partner) for my summer 2014 internship at Google was revamping the Google Translate Chrome Extension. I took an active part in the design process for the new version and implemented the UI, also working with existing code to get translation results for the new functionality while maintaining the old functionality. It was launched on 10/15/14, and reversed the negative growth trend of daily installs into strong growth (+75%). It also increased the previous 4.19 star rating on the Chrome Store into a 4.3 average rating!

  • The Brown Log

    First Prize
    "An elegant way for people to express their thoughts and ideas while doing something not so elegant. It's is an app that allows you to quickly and easily share the epiphanies that come to you while on the loo. Snap a photo of the QR code in your bathroom stall and be directed to a forum where you can post text and doodles; or, browse the thoughts that have occurred in nearby bathrooms as well."
    Created for LinkedIn Intern Hackday 2014 in a team of 5 using Angular (with MongoDB as the backend) and converted to an Android App with PhoneGap. I worked on frontend development, most notably implementing the drawing canvas feature.

  • Women's Health Toolkit Android App

    A fun little project (group of 4) for a Software Engineering course, an informational app about breastfeeding for Penn Nursing. Our first foray into Android development.

  • Mini Maya

    Demo coming soon!
    A simplified version of Autodesk Maya, a 3D computer graphics software, coded in C++ with OpenGL and Qt GUI framework. This was a 3-person group project for CIS 277 (Introduction to Computer Graphics), for which I implemented free-form deformation and procedural global transformations like twist, bend, and taper, and the import and export of OBJ files.

  • 2D Flipbook

    Demo coming soon!
    Another project from my Intro to Computer Graphics class: a 2D Flipbook implemented in C++. You can pose a character, save keyframes, interpolate between them, and playback the resulting animation.
